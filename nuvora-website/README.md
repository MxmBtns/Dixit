# Nuvora — Premium gepersonaliseerde notenpotjes

Luxueuze one-page website voor het voedingsmerk **Nuvora** (werknaam): hoogwaardige,
gepersonaliseerde notenpotjes voor bedrijven, horeca, evenementen en exclusieve geschenken.

## Branding

- **Diep zwart** als hoofdkleur — elegantie en exclusiviteit
- **Goud** als belangrijkste accentkleur — kwaliteit, luxe en vakmanschap
- **Beige/crème** — warmte, natuurlijke ingrediënten en ambacht

## Secties

Hero · Het concept (3 stappen) · Productpresentatie (Signature Nut Jar) ·
Personalisatie met interactieve preview · Voor bedrijven · Evenementen ·
Luxe verpakking · Duurzaamheid (beige) · Over ons · Premium configurator met offerteformulier

## Interactief

- Logo-upload (verschijnt live op het deksel in beide previews)
- Dekselstijl kiezen (zwart mat / goud / beige)
- Bedrijfsnaam verschijnt live op het gouden label
- Configurator: formaat, notenmix, branding, hoeveelheid + offerte-aanvraag
- Reveal-animaties, gouden shimmer, volledig responsive

## Gebruik

Volledig statisch, geen build nodig — open `index.html` in een browser
of serveer de map met bv. `npx http-server nuvora-website`.
